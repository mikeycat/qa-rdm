SETUP GUIDE: localhost Instructions - Start

========================================================================================================

1. Download source code to your local system

	•	Install heroku cli
	•	Heroku login
	•	Heroku git:clone -a app

2. Database Install Postgres: Install postgress

3. Install Dependenties (Node.js, Angular, TypeScript)

	•	Got to root of app
	•	Install npm
	•	npm install -g @angular/cli

4. Make change to your localhost database file

Note: Files to look at: package.json and src/server.ts

	•	FileName: ormconfig.json (change to your local database settings)
	•	Location of the file: root

5. Run the server

	•	Go to root of app
	•	npm run start
